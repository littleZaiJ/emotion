# 恋爱账单 (Project Clarity) — PRD v3.0
> 融合「恋爱账单」情感量化 + 「亏了么」金融终端设计理念

---

## 1. 产品综述

### 1.1 产品定位
**舔狗金融终端** — 将情感投入转化为可量化的金融数据，帮助用户做出理性决策。

- 对用户：它是「情感审计师」，通过 ROI、CPW 等核心指标帮助清醒。
- 设计隐喻：Bloomberg/同花顺终端风格，把恋爱当股票炒。

### 1.2 核心指标
| 指标 | 公式 | 含义 |
|------|------|------|
| **情感赤字 (Deficit)** | 总回馈 - 总投入 | 负数 = 在亏钱 |
| **ROI** | 总回馈 / 总投入 | 越低越危险 |
| **CPW (单字成本)** | 总投入 / 对方回复总字数 | 越高说明越贵买到一个字 |

### 1.3 用户画像
| 代号 | 特征 | 核心需求 |
|------|------|------|
| P1: ATM型 | 频繁发红包、买单，对方几乎不回礼 | 看到资金赤字，量化"被当提款机" |
| P2: 保姆型 | 出力不出钱，接送/家务/代写 | 将劳务折算为金钱，证明付出多少 |
| P3: 望夫石型 | 秒回但对方轮回，情绪随回复波动 | 量化等待成本，CPW 戳破美梦 |

## 2. 功能架构（5个核心模块）

### 2.1 底部导航
```
[ 大盘 ] [ 记一笔 ] [ 复盘 ] [ 研报 ] [ 设置 ]
```

### 2.2 模块 A：大盘仪表盘 (Dashboard)

**页面布局（从上到下）：**

1. **滚动股票行情条 (Ticker Tape)**
   - 格式：`🔴 CPW: ¥XX.X/字 | 净值: -¥XXXX | 评级: [CC级] | 警告: 流动性危机`
   - 颜色：根据健康状态切换（绿/橙/红）

2. **核心数据卡片**
   - 「情感赤字」大字：正数=绿色(涨)，负数=红色(跌)，遵循中国股市配色
   - 三个小卡片：总投入 | 总回馈 | ROI%

3. **CPW 展示区**
   - 显示：`每字成本 ¥XX.X`，附评级标签（优质股/震荡中/建议割肉/杀猪盘）

4. **近7日走势图 (K线图)**
   - 支持两种视图切换：「累计走势」面积图 / 「每日复盘」柱状图
   - 颜色：绿柱=支出(亏)，红柱=回馈(赚)（中国股市风格）

5. **等待回复计时器（大按钮，核心入口）**
   - 未计时：全宽大按钮「▶ 开始等待回复」，绿色边框
   - 计时中：显示已等待时长，背景颜色随时间变化（绿→黄→红）
   - 超过尊严阈值（默认4h）：全屏红色 Warning 状态，「尊严预警」
   - 按钮文案：计时中显示「✓ 对方回了」（点击结束，记录本次等待）

### 2.3 模块 B：记一笔 (Trade) — 底部弹窗

**两种录入类型：**

**支出模式（做空/Expense）：**
- 切换：「金钱支出」/ 「时间劳务」
- 金钱：输入金额 + 选类别（红包/礼物/餐饮/其他）+ 支出/回馈方向切换
- 劳务：滑块选时长 → 实时显示「折合价值：¥XXX」(时长×时薪)
- 可选：附加现金成本（如油费）

**回馈录入（做多/Income）：**
- 金额 + 类别
- **对方回复字数（除权/Interaction）**：只做 CPW 分母，不计入金额

**保存后：** 自动刷新仪表盘，展示最新赤字变化

### 2.4 模块 C：复盘 (History)

**两级分组列表：**
- 一级：按月份折叠（如「2026年3月」，显示当月小计）
- 二级：按日期排列，每条记录显示：类型图标 | 金额 | 时间 | 备注

**交互：**
- 左滑：删除（需确认）
- 点击：查看详情 / 编辑
- 顶部搜索：按备注/金额筛选

**底部统计：**
- 本月总支出 / 本月总回馈 / 本月净赤字

### 2.5 模块 D：首席分析师研报 (Analyst Report)

**评级体系（基于 CPW + ROI + 净值走势）：**
| 评级 | 触发条件 | 标签 | 印章颜色 |
|------|----------|------|---------|
| AAA (Prime) | CPW < 5 且 ROI > 0.8 | 强力买入 🟢 | 绿色 |
| BBB (Stable) | CPW < 20 且 ROI > 0.3 | 观望 | 蓝色 |
| BB (Junk) | CPW 20~50 或 ROI 0.1~0.3 | 谨慎增持 | 橙色 |
| CC (High Risk) | CPW > 50 或 ROI < 0.1 | 减仓止损 🔴 | 红色 |
| D (Default) | CPW > 100 或 超48h无互动 | 清算破产 💀 | 黑色 |

**特殊场景文案库：**
- 「死猫跳」：连续亏损后今日微涨 → 「技术性反弹，基本面未改善，建议谨慎」
- 「流动性危机」：本周支出 > ¥500 → 「资金链承压，检测到流动性危机」
- 「杀猪盘」：CPW > 100 → 「检测到庞氏骗局特征，建议立即卸载联系方式」

**UI 展示（研报弹窗）：**
1. 标题：「CLARITY RESEARCH — 首席分析师研报」
2. 标的代码：「LOVE-001」｜ 日期：YYYY/MM/DD
3. 评级大字 + 颜色
4. 金融黑话分析文本（英文评级 + 中文详细分析）
5. 动画：0.5s 后盖章特效（红/绿印章旋转落下）
6. 底部签名：「首席分析师 — AI · 毒舌版 / 规则引擎版」

**AI 集成：**
- 默认：规则引擎（离线，无需网络）
- 可选：接入 Claude API 生成更毒舌的个性化分析

### 2.6 模块 E：设置 (Settings)
- 我的时薪（默认 ¥50/h）
- 尊严阈值（默认 240 分钟）
- 对方名字/代号（显示在仪表盘标题）
- 数据备份/恢复
- 一键销户（清除全部数据）

## 3. 视觉规范

### 3.1 色彩体系（中国股市风格）
| 用途 | 颜色 | Hex |
|------|------|-----|
| 背景 | 纯黑 | `#000000` |
| 卡片背景 | 极深灰 | `#0D0D0D` / `#1A1A1A` |
| **支出/亏损（跌）** | **荧光绿** | **`#00C087`** |
| **回馈/盈利（涨）** | **鲜红** | **`#FF3B30`** |
| 警告 | 橙黄 | `#FF9500` |
| 主文字 | 白 | `#FFFFFF` |
| 次要文字 | 灰 | `#8E8E93` |
| 边框 | 深灰 | `#2A2A2A` |

> ⚠️ 注意：遵循中国股市「绿跌红涨」惯例（与西方相反）

### 3.2 字体
- 数字/代码：**Roboto Mono**（等宽）
- 正文/标题：**Space Grotesk**
- 研报特殊段落：系统 Serif

### 3.3 动效
- 研报盖章：旋转 + 缩放 (0.5s ease-out)
- 计时器颜色渐变：绿 → 黄 → 橙 → 红
- 数字变化：CountUp 动画
- 状态转换：200ms fade

## 4. 动态主题（ROI 状态机）


| 状态 | 触发条件 | 主色 | UI特征 |
|------|----------|------|--------|
| Level 1 双向奔赴 | ROI > 0.8 | 荧光绿 #00C087 | 圆润曲线，宽松间距 |
| Level 2 单向倾斜 | 0.3 < ROI ≤ 0.8 | 橙黄 #FF9500 | 顶部常驻 Warning 条 |
| Level 3 沉没陷阱 | ROI ≤ 0.3 | 鲜红 #FF3B30 | 进入时红色闪烁，按钮变大 |

---

## 5. 非功能性需求

- **本地优先**：所有数据存储在 Isar 本地数据库，隐私有保障
- **冷启动**：< 1.5s（随时想记账都能立刻打开）
- **字数统计**：支持中文字符计数（不计空格标点）

---

## 6. 核心指标 (North Star Metrics)

- **日均记账笔数**：验证工具价值
- **研报分享率**：验证病毒传播（用户愿意把「杀猪盘评级」发到朋友圈）
- **计时器使用频率**：验证等待焦虑场景的核心价值
